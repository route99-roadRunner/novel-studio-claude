# 통합 기획서: TS 헌터 — 세계의 규칙이 날 미소녀로 만들었다

## 스토리 요약

이세계 침식이 심화되며 세계의 규칙이 새롭게 업데이트된다. 이번 규칙은 "TS" — 성별 전환이자 회춘이자 미소녀 보정이 붙는 기이한 권능. 헌터국 소속 A급 헌터 강태현(32세)은 국가 프로젝트의 시험체로 자원하여 TS마석을 사용하고, 10세 미소녀로 변해버린다. 성인 의식은 남아있지만 감정과 충동은 아이가 되어버린 태현은 혼란 속에서도 이세계 침식에 맞서야 한다.

---

## 장르 분석 (Genre Specialist)

### 주 장르
- **현대 판타지 / 헌터물** + **TS(성전환)물** + **코미디**

### 장르 필수 요소
| 요소 | 적용 |
|------|------|
| 헌터 랭크 시스템 | S~F급, 주인공 A급 |
| 이세계 침식 / 던전 | 균열(크랙) 발생, 침식 등급제 |
| 세계 규칙 업데이트 | 시스템 메시지로 전달, 강제 적용 |
| TS 미소녀 보정 | 외모/체형 자동 보정, 젊어짐 |
| 갭 모에 | 아이 외모 ↔ A급 전투력 |

### 활용할 트로프
1. **갭 모에**: 귀여운 외모 vs 무시무시한 전투력 (핵심 코미디 장치)
2. **오해와 반전**: 주변인들이 아이 취급 → 전투 시 경악
3. **시스템 메시지**: 세계 규칙 업데이트를 팝업창 스타일로 표현
4. **내면 독백 갭**: 내면은 아저씨인데 행동은 아이 (부분 퇴행으로 인한 괴리)

### 피할 클리셰
- 과도한 성적 묘사 (10세 외모이므로 절대 금지)
- 단순 파워판타지 (능력이 아닌 상황 코미디가 핵심)
- TS를 단순 개그 소재로만 소비 (정체성 혼란의 진지한 면도 간간이)

## 캐릭터 프로필 (Character Writer)

### 주인공: 강태현 / 강하나

#### 변신 전: 강태현 (32세, 남)
- **직업**: 헌터국 특수대응과 A급 헌터 (공무원 7급 상당)
- **외모**: 176cm, 보통 체격. 정장 위에 헌터국 점퍼를 걸친 공무원 스타일. 눈에 띄지 않는 평범한 인상
- **성격**:
  - 성실하고 책임감 강함 ("시키면 합니다" 타입)
  - 감정 표현 서툴고 무표정
  - 의외로 소심한 면 (상사한테 할 말 못하는 편)
  - 독신, 자취, 편의점 도시락이 주식
- **전투 스타일**: 강화계 — 신체 강화 특화, 화려하진 않지만 견고하고 효율적
- **습관**: 보고서 양식에 집착, 모든 걸 항목별로 정리하려 함

#### 변신 후: 강하나 (외견 10세, 여)
- **외모**: 131cm, 은발에 붉은 눈 (마력 변환 부작용). 인형 같은 얼굴. 교복 사이즈가 없어서 임시로 아동복 착용
- **성격 변화 (부분 퇴행)**:
  - 성인 의식과 지식은 유지됨 (보고서 쓸 수 있고, 전술 판단 가능)
  - 그러나 감정 제어력이 급락: 사탕에 솔직해지고, 무서우면 울고, 화나면 뺨 부풀림
  - 집중력 저하: 회의 중 졸음, 긴 보고서 읽다가 딴짓
  - 어미가 가끔 아이 말투로 샘 ("~인 거예요!", "싫어요!")
- **전투 능력**: A급 유지. 오히려 작은 체구가 기동성에 유리. 단, 리치 부족
- **특징적 습관**: 화날 때 발 동동 구르기, 보고서에 무의식적으로 낙서, 높은 곳을 못 올라감

### 조연: 박준혁 (38세, 남)
- **직업**: 헌터국 특수대응과 과장, S급 헌터
- **역할**: 태현의 직속 상사이자 TS 프로젝트 책임자
- **성격**: 냉정한 듯하지만 부하 챙기는 타입. 태현이 변하고 나서 '딸바보' 모드 발동하는 자신에게 당황
- **핵심 대사**: "강 요원, 보고서 마감이... 아니, 하나야, 배고프지 않아?"

### 조연: 김소라 (28세, 여)
- **직업**: 헌터국 연구원, TS마석 연구 담당
- **역할**: TS 현상을 분석하는 과학자. 태현(하나)을 관찰 기록
- **성격**: 학구적이고 냉정하지만, 하나의 귀여움에 점점 무장해제
- **핵심 대사**: "객관적 관찰을 위해 거리를 유지해야... 세상에 이 볼 좀 봐요."

## 세계관 설정 (Setting Writer)

### 세계 규칙 시스템
- 이세계 침식이 시작된 이후 "세계의 규칙"이 주기적으로 업데이트됨
- 업데이트는 시스템 메시지(반투명 팝업창)로 전 인류에게 동시 통보
- 과거 업데이트 이력:
  - 1차: 마나 각성 (헌터 탄생)
  - 2차: 던전(균열) 발생
  - 3차: 랭크 시스템 부여
  - **4차 (현재)**: TS 권능 추가

### TS 규칙 상세
```
◆ 세계 규칙 업데이트 v4.0 ◆

[TS마석] 사용 시 다음이 적용됩니다:
1. 성별 전환 (남→여 / 여→남)
2. 외모 최적화 보정 (최저보정: 미소녀/미소년)
3. 연령 회춘 (사용자 기준 최적 전투 연령으로 조정)
4. 체형 자동 관리 (유지보수 불필요)
5. 정신 동조 (외견 연령에 부분 동기화)

※ 해제 조건: 미정
※ 전투 능력: 유지
```

### 헌터국 (국가기관)
- 정식 명칭: 대한민국 이세계침식대응청 (약칭 헌터국)
- 위치: 서울 용산구 (구 국방부 자리)
- 구조: 특수대응과, 연구과, 행정과, 의료과
- TS 프로젝트: "프로젝트 트랜스폼" — TS마석의 군사적 활용 가능성 검증

### 이세계 침식
- 균열(크랙)에서 마수가 출현
- 침식 등급: 청색(경미) → 황색(주의) → 적색(위험) → 흑색(재난)
- 최근 황색~적색 빈도 증가 → 4차 규칙 업데이트의 원인으로 추정

## 스토리 구조 (Story Writer)

### 플롯 아크
- **시작점**: 성실한 A급 헌터 강태현의 평범한 일상
- **촉발 사건**: 4차 세계 규칙 업데이트 → TS 프로젝트 시험체 자원
- **상승**: TS 변신 후 혼란, 주변 반응, 능력 테스트
- **클라이맥스**: 적색 균열 긴급 출동 — 10세 미소녀의 A급 전투
- **해소**: 새로운 정체성의 시작, 해제 조건 떡밥

### 챕터별 상세

#### Chapter 1: 세계가 또 업데이트됐다
**목표**: 세계관 소개 + 주인공 소개 + TS 변신까지
**감정선**: 일상(30%) → 긴장(60%) → 충격/코미디(100%)

- **장면 1**: 강태현의 출근 일상 (헌터국 공무원의 평범한 아침)
- **장면 2**: 4차 세계 규칙 업데이트 발동 — 시스템 메시지 팝업
- **장면 3**: 긴급회의 — TS마석 발견, 프로젝트 트랜스폼 설명
- **장면 4**: 태현이 시험체로 자원하는 이유 (성실한 공무원의 사명감 + 약간의 호기심)
- **장면 5**: TS마석 사용 → 변신 장면 (빛에 휩싸이고 눈을 뜨니 시점이 낮아져 있다)
- **클리프행어**: 거울에 비친 자신 — 은발 붉은 눈의 10세 미소녀

#### Chapter 2: 이건 보고서에 어떻게 쓰죠
**목표**: 변신 후 혼란 + 부분 퇴행 코미디 + 능력 테스트
**감정선**: 혼란(80%) → 코미디(100%) → 약간의 불안(40%)

- **장면 1**: 변신 직후 패닉 — 목소리, 키, 체형 확인. 보고서를 쓰려는데 손이 작아서 키보드가 안 맞음
- **장면 2**: 김소라 연구원의 관찰 기록 시작 — 신체 검사, 마력 측정
- **장면 3**: 정신 퇴행 발현 — 회의 중 졸림, 사탕에 솔직해짐, 화나면 볼 부풀림
- **장면 4**: 박준혁 과장의 '딸바보' 모드 발동 — 프로답게 대하려는데 실패
- **장면 5**: 능력 테스트 — 전투 시뮬레이션에서 A급 실력 발휘. 단, 테스트 끝나고 "피곤해요..." 하고 쓰러져 잠듦
- **클리프행어**: 적색 균열 경보 발령 — "강... 하나 요원, 출동입니다"

#### Chapter 3: 미소녀가 전장에 서면
**목표**: 첫 실전 + 갭 모에 폭발 + 여운
**감정선**: 긴장(70%) → 전투 고양(100%) → 코미디(80%) → 여운(50%)

- **장면 1**: 출동 준비 — 전투복이 사이즈가 없어서 급조. 헬멧이 커서 흘러내림
- **장면 2**: 현장 도착 — 다른 헌터들의 반응 ("애가 왜 여기에?")
- **장면 3**: 전투 시작 — 마수와 교전. A급 전투력 발휘, 주변 경악
- **장면 4**: 전투 중 정신 퇴행 발동 — 무서운 마수에 잠깐 울먹이다가 분노로 전환, 더 강해짐
- **장면 5**: 전투 종료 후 — 보고서를 쓰는데 중간에 낙서가 섞임. "오늘 전투 요약: 이겼다. 무서웠다. 사탕 먹고 싶다."
- **엔딩**: 박준혁 과장이 사탕을 건네며 "수고했다, 하나야." / 하나가 사탕 받아들며 볼이 붉어짐 / "...감사합니다, 과장님." / 시스템 메시지: [TS 해제 조건: ???] → 후속 떡밥

## 핵심 코미디 장치

1. **보고서 갭**: 성인 형식의 보고서를 쓰려는데 중간에 아이 말투/낙서가 섞임
2. **딸바보 전염**: 상사/동료가 하나를 대하면서 점점 무장해제
3. **전투 갭**: 귀여운 외모로 A급 전투 → 주변 인물들 경악
4. **자판기 갭**: 버튼에 손이 안 닿아서 점프, 높은 선반 못 닿음
5. **내면 독백**: 「나는 32세 A급 헌터 강태현이다. 사탕 따위에... 맛있다.」

## 문체 가이드

- **화자**: 3인칭 제한 시점 (태현/하나 중심)
- **내면 독백**: 「 」 꺽쇠 사용, 성인 말투 유지
- **시스템 메시지**: `[시스템]` 접두사, 고딕체/볼드 느낌
- **대화**: 하나의 말투는 존댓말 기반이지만 감정적일 때 아이 말투 샘
- **전투 묘사**: 간결하고 임팩트 있게 (웹소설 스타일)
- **코미디 타이밍**: 진지한 장면 직후 갭으로 웃음

## 창작 시 주의사항

1. 10세 외모 캐릭터이므로 성적 묘사 완전 배제
2. TS를 단순 개그만이 아닌, 정체성 혼란의 씨앗으로도 활용
3. 부분 퇴행은 "귀여움"을 위한 장치이되, 전투 판단력은 유지
4. 시스템 메시지는 과하지 않게 — 필요한 순간에만 등장
5. 웹소설 연재를 의식하되 단편 완결성 확보 (후속 떡밥은 열린 결말로)
